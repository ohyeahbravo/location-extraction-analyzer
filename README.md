# location-extraction-analysis

## Requirements
Recent versions of
* Node.js
* npm

## Setup
```
npm install
```

## Start the Server
```
node server.js
```

## Start the Vue App
In the separate terminal,
```
npm run serve
```
then open http://localhost:8080 in a browser.

[Configuration Reference](https://cli.vuejs.org/config/) for more Vue configurations.
